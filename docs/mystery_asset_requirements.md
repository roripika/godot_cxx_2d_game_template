# Mystery Demo 必須グラフィックアセット一覧

以下のリストは、Mystery デモ（アドベンチャー/逆転裁判風ゲーム）の現在のシナリオ（`mystery.yaml`）および実装において必要となるグラフィックアセットの一覧です。
アセット作成担当者への依頼用リストとして活用してください。

---

## 🖼️ 1. 背景画像 (Backgrounds)
各シーンのベースとなる背景絵です。必要サイズはゲームの解像度（例えば 1920x1080 や 1152x648 など）に合わせてください。

| アセット名 | 用途・表示シーン | イメージ・要件 |
| :--- | :--- | :--- |
| **探偵事務所 (Office)** | プロローグ（導入）、推理パート、結末シーン | 古びた探偵事務所、デスク、資料の山など。 |
| **廃倉庫 (Warehouse)** | 調査パート、対決（法廷）パート | 薄暗い倉庫、木箱、事件現場の雰囲気。 |
| **エンディングの一枚絵 (Ending)** | グッドエンド（事件解決）、バッドエンド | ※任意。<br>専用絵がなければ事務所や倉庫の背景を流用、あるいは暗転でも可。 |

---

## 👤 2. キャラクター立ち絵 (Character Portraits)
会話システム（DialogueUI）や対決パートで表示するキャラクターのバストアップ（または全身）の立ち絵画像です。
**※可能であれば「通常」「驚き（ダメージ）」「怒り/焦り」などの表情差分があると、対決パート（ゆさぶる/つきつける）の演出が映えます。**

| キャラクター名 | 役割・表示シーン | イメージ・要件 |
| :--- | :--- | :--- |
| **探偵 (Detective)** | 主人公、プレイヤーキャラクター。全編。 | プレイヤーの分身。 |
| **所長 (Boss)** | 探偵事務所の上司。<br>プロローグと、推理パートでプレイヤーにクイズを出題する役。 | 威厳がある、あるいは少しとぼけた上司など。 |
| **倉庫管理人 / ネズミの証人 (Rat Witness)** | 倉庫の管理人。<br>対決パートで証言を行い、プレイヤーに尋問される相手。 | 少し怪しい雰囲気の小悪党イメージ。 |

---

## 🔍 3. 証拠品アイコン (Evidence Icons)
インベントリ UI や対決パートの「つきつける」で表示するための小さめのアイコン画像です。
推奨サイズ: 64x64 または 128x128 程度（正方形に近い比率）

| アイテム名 | 用途・表示シーン | イメージ・説明 |
| :--- | :--- | :--- |
| **エクトプラズム (Ectoplasm)** | 調査パートで取得。対決パートの決定打。 | 幽霊が出た証拠となる光る粘液。 |
| **血の足跡 (Footprint)** | 調査パートで取得。対決パートで使用。 | 人間のものではない、巨大な血のついた足跡。 |
| **破れたメモ (Torn Memo)** | 調査パートで取得。対決パートで使用。 | 「午前3時に...」と書かれた破れた紙切れ。 |

---

## 🎨 4. UI パーツ（※任意、世界観強調用）
標準のパネルやボタンで代用可能ですが、世界観を強調したい場合は以下の UI アセットもあると完成度が上がります。

| UI パーツ | 用途 |
| :--- | :--- |
| **メッセージウィンドウ枠** | 会話テキストを表示する下部の枠。 |
| **アイテムスロット枠** | 証拠品を並べるインベントリ画面での背景枠。 |
| **ライフ（HP）表示アイコン** | ハートアイコン、あるいはペナルティを表すマークやゲージ。 |
| **対決用コマンドボタン** | 「ゆさぶる (Press)」「つきつける (Present)」に使う専用ボタン画像。 |
