◆ pop_scene()

void karakuri::SceneFlow::pop_scene ( )

現在のシーンを破棄し、スタックに保持していた直前のシーンを復元する。

現在シーンを queue_free() し、suspend 中のノードを add_child() で再接続する。 スタックが空の場合は何もしない。

◆ push_scene()

void karakuri::SceneFlow::push_scene ( const godot::String &  path,
const godot::Dictionary &  params = godot::Dictionary() 
)

現在シーンをメモリ上に保持したまま新しいシーンへ移る。

現在のノードを SceneTree から remove_child()（= 凍結）し、 新しいシーンを即座に instantiate() して add_child() する。 pop_scene() で凍結したノードをそのまま復元できる（リロード不要）。
